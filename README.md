# Malaysian Dictionary

## Introduction

Although the Malaysian language is similar to Indonesian, there are important differences that need to be understood carefully.

This app is designed to help users learn Malaysian by providing explanations of vocabulary and grammar in a clear and practical way.

## Main Features

* Search and save Malaysian words
* View saved vocabulary records
* Use a vocabulary notebook to help memorize saved words

## How to Use

1. Download the app.
2. Open and run the app.
